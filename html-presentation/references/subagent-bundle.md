# Subagent Reference Bundle
Condensed reference for slide-building subagents. Mandatory rules in prompt block take precedence.

---

## Presentation Runtime

Shell provides: `#nav` pill (fixed bottom-right, ← → arrows, counter, notes hint), slide transitions (opacity-based, `position:absolute;inset:0;opacity:0` → `.slide.active` sets `opacity:1`), speaker notes (`N` = popup window, `B` = bottom panel), keyboard nav (Arrow keys, `N`, `B`).

`<div class="counter"></div>` before `#nav` — required by `generate_qr_appendix.py`. Do not remove.

Never toggle `display:none`/`display:flex` on slides — breaks CSS transitions.

Speaker notes: `<div class="speaker-notes">...</div>` inside slide div. Stage directions in brackets: `[Pause here]`, `[Click to advance]`.

`show()` signature:
```js
function show(n) {
  slides[current].classList.remove('active');
  current = (n + slides.length) % slides.length;
  slides[current].classList.add('active');
  document.getElementById('curr').textContent = current + 1;
}
```

Per-slide JS: extend `show()` in `<script>`. These classes auto-trigger via shell (no JS needed): `.anim-left`, `.anim-right`, `.scale-pop`, `.draw-line`, `.progress-fill`, `.ring-fill`, `.counter`, `.typewriter`.

## Slide Structure

| Slide | Purpose |
|-------|---------|
| 1. Title | Big title, subtitle, brand |
| Agenda | Card grid of 3–6 sections (not counted toward max) |
| BLUF | 3-card key takeaways (not counted toward max) |
| 2. Setup | Frame problem/context |
| 3-8. Core | One idea per slide, each with visual component |
| 9. Evidence | Stats, quotes, case studies |
| 10. Takeaway | Summary, CTA, next steps |

Complexity scaling: simple 5–7, moderate 10–15, deep-dive 20–25. Max 30. Exec mode: 7-slide cap, no Agenda, BLUF embedded in s2.

Presenter slide: after Title, before Agenda. Not counted. Headshots: `border-radius:50%;object-fit:cover`. Max 9. See visual-components Presenter Slide pattern.

Agenda: numbered card grid (2-3 cols). Each card: accent icon (`clamp(1.5rem,2.5vw,2rem)`), bold title (`clamp(0.9rem,1.4vw,1.2rem)`), descriptor (`clamp(0.75rem,1.1vw,1rem)` secondary). Heading = small uppercase `h3` with accent, NOT `h2`. Stagger animations.

BLUF: 2–4 cards, each: headline (bold white) + elaboration (secondary) + accent icon. Eyebrow `h3`.

## Content Rules

1. One idea per slide
2. ~30 words/slide (guideline, not hard limit)
3. Every slide needs visual component — no text-only slides
4. Consistent accent color throughout
5. No bullet dumps — use Icon+Label List
6. Big text beats small text
7. Fill viewport: `max-width:min(1200px,92vw)`, `clamp()` padding. Balanced density, not clutter
8. Material Symbols not emojis. Endpoint: `css2?family=Material+Symbols+Rounded`. Class: `material-symbols-rounded`
9. CSS animations on primary content. No infinite loops on static. Respect `prefers-reduced-motion`
10. CSS illustrations, SVG shapes, gradients for visuals. All inline. Total <2MB, images <500KB
11. No double bullets: icon/symbol `<li>` → parent `<ul>` needs `list-style:none;padding:0;`
12. Left-align lists: `text-align:left` on `<ul>`/`<ol>`. Explicit when inside centered container
13. Links: `target="_blank" rel="noopener"`. Style: `a{color:inherit;text-decoration:none;border-bottom:1px solid var(--accent);transition:color 0.2s;} a:hover{color:var(--accent);}`

**Conditional rules (skip if N/A):**
14. QR appendix: auto-generated by `generate_qr_appendix.py` if external links exist
15. Architecture diagrams: full-width for 4+ nodes. Min readable: node 36px, label font 12, arrow 16px
16. SVG hard rules: no overlapping rects (10px min gap), no rotated text, max-height≥58vh, viewBox height = lowest element + 20px, MUST use `svg_calc.py`
17. Removing stat card → update `repeat(N,1fr)`
18. Wrong 3× worse than unknown — omit unconfirmed facts
19. SVG boxes sized to fit text — run `svg_calc.py textbox`. Default ≥200px for long labels. 24px padding
20. Exec mode: declarative assertion titles, s2 = Executive Summary (CTA/Stat, no bullets), 7-slide cap, skip Agenda

## HTML Output Spec

Single self-contained HTML. One external CDN: Material Symbols Rounded.

Slide structure:
```html
<div id="sN" class="slide">
  <div class="slide-inner">[content]</div>
  <!-- optional: <div class="speaker-notes">...</div> -->
</div>
```
`.slide-inner` = padding `clamp(1.65rem,4.4vh,3.85rem) clamp(1.65rem,5.5vw,4.4rem)`, `max-width:min(1320px,95vw)`. Exception: ANY slide may place ambient glow orb divs as direct children BEFORE `.slide-inner` (add `overflow:hidden` to the `.slide` div, use percentage-based coords, set `z-index:0` on orbs and `z-index:1` on `.slide-inner`). NEVER place orbs inside `.slide-inner`.

CSS vars: `--bg:#0a0a0a`, `--text:#fff`, `--secondary:#a0a0a0`, `--card:#1a1a1a`, `--border:#2a2a2a`, `--accent` per deck.

Typography (relative units only, no px for font-size):
- H2: `clamp(2.75rem,4.4vw,4.4rem)`
- H3 eyebrow: `clamp(0.935rem,1.32vw,1.1rem)`
- Body/li: `clamp(1.1rem,1.76vw,1.65rem)`
- Stats: `clamp(4.4rem,7.7vw,6.6rem)` accent
- Code: `clamp(0.96rem,1.32vw,1.24rem)` mono
- Captions: `clamp(0.825rem,1.1vw,0.99rem)`

Spacing: `vh`/`vw`/`rem`. `px` only for border-width, border-radius, letter-spacing.

Nav: arrow keys, counter bottom-right. `<div class="counter"></div>` before nav block. JS init: `document.getElementById('total').textContent=slides.length;` before `show(0)`.

Transitions: opacity + pointer-events. Never `display:none/flex`.

Speaker notes: `<div class="speaker-notes">` inside slide. `N`=popup, `B`=panel. Bracketed text = stage directions.

## Architecture Diagram / Inline SVG Rules

Connectors: `stroke="var(--accent)"` or bright hex. NEVER dark gray/`var(--border)`.

Arrow markers: `markerUnits="userSpaceOnUse"`. Standard 10x7, `refX="10" refY="3.5"`. Unique ID per SVG (e.g. `arr7`).
```html
<defs>
  <marker id="arrN" markerWidth="10" markerHeight="7" refX="10" refY="3.5" orient="auto" markerUnits="userSpaceOnUse">
    <polygon points="0 0, 10 3.5, 0 7" fill="var(--accent)"/>
  </marker>
</defs>
```

No inline icon SVGs — use `<span class="material-symbols-rounded">` instead.

viewBox tight to content: bottom gap ≤12% of height. Set height = max_content_bottom + 20px.

Pattern 10 CSS (add per slide):
```css

@keyframes blinkCaret { 0%,100%{border-color:transparent} 50%{border-color:var(--accent)} }

@media(prefers-reduced-motion:reduce){.typewriter{width:100%;animation:none;}}
```

Pattern 14 JS (add once per deck):
```javascript
function twTypewrite(el, html, speed = 6) {
  let total = 0;
  for (let i = 0; i < html.length; ) {
    if (html[i] === '<') { i = html.indexOf('>', i) + 1; } else { total++; i++; }
  }
  let shown = 0;
  const timer = setInterval(() => {
    if (shown > total) { clearInterval(timer); el.innerHTML = html; return; }
    let out = '', count = 0;
    for (let i = 0; i < html.length && count < shown; ) {
      if (html[i] === '<') { const end = html.indexOf('>', i); out += html.slice(i, end + 1); i = end + 1; }
      else { out += html[i]; count++; i++; }
    }
    el.innerHTML = out + '<span class="tw-cursor"></span>';
    shown++;
  }, speed);
}
```

Wiring: store `innerHTML` at page init. On enter: `.tw-pending` → 650ms → lock `offsetHeight` → clear → `twTypewrite`. On leave: restore `innerHTML`, release height. `prefers-reduced-motion`: skip animation, show SOURCE directly.

Rules: durations 0.3–1.5s. `ease`/`ease-out` for enters. No infinite loops unless live state. Must be readable if animations fail.

## Graphics Embedding

All graphics inline. Use `scripts/run_script.py <script.py> [args]` for all operations.

Tokens (prefer highest in table):
| Token | For | Result |
|-------|-----|--------|
| `{{SNOWFLAKE_LOGO}}` | Snowflake logo only | inline `<svg>` |
| `{{SVG_INLINE:path}}` or `{{SVG_INLINE:path\|css}}` | User SVGs | inline `<svg>` |
| `{{LOGO_INLINE:path}}` or `{{LOGO_INLINE:path\|css}}` | Logos (skips geometry check) | inline `<svg>` |
| `{{IMG:path}}` or `{{IMG:path\|max-px}}` | Raster images | `data:` URI |

`{{SVG_INLINE}}` > `{{IMG}}` for SVGs (no base64, inherits CSS vars, styleable).

NEVER read base64 into context. NEVER paste base64 into HTML. NEVER copy SVG `<path>` manually. Use placeholder tokens — `embed_image.py` handles injection.

Image CSS: `border-radius:12px`, `max-width:400px;max-height:300px`, `display:block;margin:0 auto`, `object-fit:contain`.

Scripts: `resize_image.py`, `screenshot_to_slide.py`, `svg_optimize.py`, `color_swap_svg.py`, `embed_image.py`, `insert_presenter.py`.

Shell safety: NEVER heredoc or multi-line `python3 -c`. Write to file first, then run.
